Study Catalog
Created 7 October 2017


About the app
Study Catalog is about keeping track of interesting courses and reading materials by major topics.

To run the app
Run application.py on the vagrant machine, then load http://localhost:8000/catalog/ on browser to see the app. Gmail is required to be able to log on and create/edit/delete resource items.

More about the Catalog folder
1. database_setup.py is the resource data.db configuration file
2. styles.css is stored under /static folder
3. see /templates for all html files
